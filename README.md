# NominaPy

NominaPy es una app MVC que Administra la Nomina de Trabajadores de tu Empresa.

## Descarga

Descarga este repositorio mediante CLI:

```bash
git clone https://github.com/bluuscript/app.git
```

## Uso

```bash
/app/ python3 main.py
```
